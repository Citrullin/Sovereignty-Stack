# Koral

Koral is the operation from Nordic Tech Labs to buy used thin clients, put sovereign software on it, and rent it out to people for a low price. The sovereign stack basically. With encrypted backups and replacement in case of emergencies. 

There are two demos in the demo directory. One about the general workflow etc. in Nextcloud, ERPNext etc.
The second demo (OICD_SIWE) is a demo about the flow to log in with your wallet. (Sign in with Ethereum)
This OIDC Request is Relayed and fulfilled by the wallet sign request. Linking OIDC backend auth to wallet.
Enabling new innovative business models by combining blockchains and Federated Open Source Software.
