# Microblocks and LDraw Files

One thing to mention upfront: Not all LDraw CAD programs are happy with Microblocks.
Even though Nanoblocks, Loz etc. are quite popular in Asia, it's hard to find decent software 
that supports it basically.
In order to have a decent experience desining the sets, I recommend to use MLCAD.
Even though it's quite old, Windows only and looks like it hasn't been updated in years, it's still 
quite usable and I have barely encountered any bugs. 

In web viewer the different layout size doesn't cause any issues.
