# A2A Telemetry & Verification Logs

[← Back to Root](../../README.md)

## Overview

The A2A telemetry system allows autonomous agents to audit the repository
without human intermediary. The JSONL log tracks commit-to-invariant
mappings over time. The chart_telemetry.py script visualizes verification
velocity for the Auditor Agent.
