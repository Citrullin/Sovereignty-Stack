# Wahfare — Branded Crypto Native Cash Disc

Wahfare Hits is a branded product line for the Sovereignty Stack's physical
**Crypto Native Cash** instrument: a small, fun, collectible NFC/RFID disc
designed for everyday P2P value transfer.

Think of it as a crypto banknote in disc form. It implements the **Banknote Mode**
of the Tiny Disc: value lives on the physical object, and possession equals
ownership. No app required for basic transfers.
