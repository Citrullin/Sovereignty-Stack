# Bit.block

Bit.block are smart microbricks if you will. They can carry electricity and data. And be stacked together to form larger structures.
Each one can also act as a pin and attach to a breadboard. For faster prototyping and hacking around.
They also may include sensors or actuators to interact with the physical world.

[Prototypes](prototypes): This directory contains pictures and videos of the bit.block pototypes. Mixed in type (PoC, v1 etc.)
[Board](board): Contains the KiCAD files to print the PCB for bit.blocks
[3D prints](3D-prints): Contains all 3D print files for the Astronaut on the Moon set.

[3D Web Emulation](3D-Web-Emulation): Contains a Three.js project to view and interact with the 3D astronaut on the moon set in the browser.
