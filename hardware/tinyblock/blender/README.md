# Diamond blocks in Blender

In order to use Tinyblocks or similar products in Blender (LDraw), you need ImportLDraw installed.

## Installation

Download the addon from [Github](https://github.com/TobyLobster/ImportLDraw) and install it in Blender.
Details on installation and usage can be found in the repository.
In order to use LDraw files on the web, it's recommended to [generate WebGL ready files](https://docs.blender.org/manual/en/latest/addons/import_export/scene_gltf2.html) (.glb).

## Future work
